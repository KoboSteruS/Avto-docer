# Система статей/блога

Система для публикации статей в стиле Telegram-канала с поддержкой изображений и видео.

## Возможности

- ✅ Создание статей через админку Django
- ✅ Поддержка изображений (главное изображение)
- ✅ Поддержка видео (YouTube, Vimeo, прямые ссылки)
- ✅ HTML-форматирование в содержимом
- ✅ Счетчик просмотров
- ✅ Публикация/снятие с публикации
- ✅ Красивый дизайн в стиле Telegram-канала
- ✅ Адаптивная верстка

## Как использовать

### 1. Создание статьи в админке

1. Зайдите в админку: `/admin/`
2. Найдите раздел **"Статьи" → "Статьи"**
3. Нажмите **"Добавить статью"**

**Заполните поля:**
- **Заголовок**: Название статьи
- **URL-слаг**: Автоматически генерируется из заголовка (можно изменить)
- **Опубликовано**: Включите, чтобы статья была видна на сайте
- **Содержимое**: Текст статьи с HTML-форматированием
- **Главное изображение**: Загрузите изображение (будет показано вверху статьи)
- **URL видео**: Ссылка на видео (YouTube, Vimeo или прямой файл)

### 2. Поддержка HTML в содержимом

В поле "Содержимое" можно использовать HTML-теги:

```html
<h2>Заголовок</h2>
<p>Обычный текст параграфа.</p>
<p><strong>Жирный текст</strong> и <em>курсив</em>.</p>

<h3>Подзаголовок</h3>
<ul>
    <li>Пункт списка 1</li>
    <li>Пункт списка 2</li>
</ul>

<p>Можно вставлять изображения:</p>
<img src="https://example.com/image.jpg" alt="Описание" />

<p>И ссылки:</p>
<a href="https://example.com">Текст ссылки</a>
```

### 3. Поддержка видео

#### YouTube
```
https://www.youtube.com/watch?v=VIDEO_ID
или
https://youtu.be/VIDEO_ID
```

#### Vimeo
```
https://vimeo.com/VIDEO_ID
```

#### Прямая ссылка на видео файл
```
https://example.com/video.mp4
```

**Важно:** Если указано и изображение, и видео, будет показано видео.

### 4. Структура страниц

- **Список статей**: `/stati/` - все опубликованные статьи
- **Детальная страница**: `/stati/slug-статьи/` - полная статья

### 5. Навигация

Ссылка "СТАТЬИ" добавлена в главное меню сайта (десктоп и мобильная версия).

## Примеры использования HTML

### Заголовки
```html
<h2>Основной заголовок</h2>
<h3>Подзаголовок</h3>
```

### Форматирование текста
```html
<p><strong>Важный текст</strong></p>
<p><em>Курсив</em></p>
<p>Обычный текст с <br />переносом строки</p>
```

### Списки
```html
<ul>
    <li>Элемент списка 1</li>
    <li>Элемент списка 2</li>
</ul>

<ol>
    <li>Нумерованный элемент 1</li>
    <li>Нумерованный элемент 2</li>
</ol>
```

### Изображения
```html
<img src="https://example.com/image.jpg" alt="Описание изображения" />
```

### Ссылки
```html
<a href="https://example.com">Текст ссылки</a>
```

### Цитаты
```html
<blockquote>
    Цитата или важная информация
</blockquote>
```

## Стилизация

Статьи автоматически стилизуются:
- Заголовки выделены белым цветом
- Подзаголовки с красным акцентом
- Текст читаемый, с хорошим межстрочным интервалом
- Изображения с закругленными углами
- Видео адаптивные (16:9)

## Админка

В админке доступны:
- Превью изображения
- Превью видео (для YouTube/Vimeo показывается embed)
- Массовые действия (публикация/снятие с публикации)
- Поиск по заголовку и содержимому
- Фильтры по статусу публикации и дате

## Миграции

Миграции уже применены. Если нужно пересоздать:

```bash
python manage.py makemigrations articles
python manage.py migrate articles
```
